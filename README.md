# LedFx

## Command Line Interface
```
% go run cmd/ledfx/cli.go -h 
NAME:
   LedFx - A CLI for LedFx

USAGE:
    [global options] command [command options] [arguments...]

VERSION:
   0.0.1

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
```
